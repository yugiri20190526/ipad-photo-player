# PhotoPlayer

iPad の写真アプリと同じ機能を持つ WEB アプリケーション。

## 機能

- **グリッド表示**: 写真・動画をサムネイル一覧で表示
- **詳細ビュー**: フルスクリーンで写真・動画を表示
- **スライドショー**: 3秒間隔で自動進行
- **スワイプ操作**: 詳細ビューで前後に移動（デスクトップボタンも対応）
- **フォルダ選択**: 複数のファイルを選択して一度に追加可能
- **IndexedDB 保存**: 選択したファイルをブラウザに保存して高速表示
- **モバイル対応**: iPad・iPhone でも快適に使用可能

## 使い方

1. 「追加」ボタンをタップ
2. 表示したい写真・動画ファイルを選択（複数選択可能）
3. グリッド表示からサムネイルをタップして詳細ビューを開く
4. 詳細ビューでボタンやスワイプで操作
   - 左右ボタン: 前後に移動
   - 再生ボタン: 動画の再生・一時停止
   - スライドショー: 自動進行を開始/停止

## デプロイ

GitHub Pages にデプロイする際の手順（ipad-video-player と同様）：

```bash
git add .
git commit -m "Update PhotoPlayer"
git push origin main
```

CI により自動的にビルド ID が更新されます。

## 技術スタック

- HTML5 + CSS3 + Vanilla JavaScript
- IndexedDB（ブラウザローカルストレージ）
- Service Worker（オフライン対応）
- File API（ファイル選択）
